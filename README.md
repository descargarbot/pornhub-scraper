# pornhub scraper
A Python class to download videos from PornHub

<h2>dependencies</h2>
<code>Python 3.9+</code><br>
<code>sudo apt-get install -y proxychains4 ffmpeg</code><br>
<code>pip install requests</code><br>
<code>pip install requests[socks] # para SOCKS5</code><br>
<code>pip install curl-cffi</code><br>
<code>pip install phantomjs-binary # + chmod +x al binario</code><br>
<br>

<h2>usage</h2>
<code>python3 ph_scraper_sock5_nvpn.py "PH_URL"</code>
